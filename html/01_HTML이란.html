h(heading)태그는 제목을 작성할 때 사용하는 태그입니다.

- p(paragraph)태그는       문단을 작성할 때 사용하는 태그입니다.
- HTML은 HyperText Markup Language의 약자입니다.
- 웹 페이지의 생김새를 묘사하는 마크업 언어입니다.
- 웹 브라우저는 HTML에 작성된 내용을 보고 화면에 어떻게 그려야할지 판단합니다.
- 웹 브라우저의 종류로는 Chrome, Safari, Whale, Edge, Internet Explorer, FireFox 등이 있습니다.
- 웹 브라우저가 HTML의 내용을 보고 화면에 그리는 작업을 렌더링(rendering)이라고 합니다.
