- 자바스크립트 사용하기

<a> 태그 사용하기

- a 태그의 href 속성에 원하는 페이지의 경로를 적어서 사용

- 자세한 내용을 알고 싶으시면 여기를 클릭하세요!
- 여기(./naver.com)
- 여기(./www.naver.com)
- 여기(/naver.com)
- 여기(//naver.com)

웹에서 사용할 수 있는 경로의 종류(./, /, //)

- https://comic.naver.com/webtoon/list?titleId=814543가 있을 때
- 해당 위치에서 ./naver.com은 https://comic.naver.com/webtoon/naver.com으로 간다.(가장 마지막 위치에 있는 /에서 해당 경로를 찾는다.)
- 해당 위치에서 /naver.com은 https://comic.naver.com/naver.com으로 간다.(가장 처음 위치에 있는 /에서 해당 경로를 찾는다)
- 해당 위치에서 //naver.com https://naver.com으로 간다.(프로토콜 다음에 있는 //에서 해당 경로를 찾는다.)

같은 경로의 다른 페이지로 링크 걸기

02
03
04

해당 페이지의 특정 부분으로 링크 걸기

- #id 속성 값을 사용해 페이지의 특정 부분으로 링크를 걸 수 있다.
챕터1
챕터2

자바스크립트를 사용해 링크 걸기

- 자바스크립트로는 꼭 a 태그가 아니더라도 링크를 걸 수 있다.
