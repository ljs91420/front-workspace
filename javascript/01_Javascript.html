  • 웹 브라우저가 해석하여 동작하는 프로그래밍 언어
  • HTML은 웹 페이지의 뼈대를 담당하고 CSS는 웹 페이지의 외관을 담당하고 JS는 웹 페이지의 동작을 담당한다.

Javascript 적용 방식 3가지

  • inline : HTML 속성(attribute)에 추가하기
  • internal : 현재 페이지의 <script> 태그 내부에 적용하기
  • external : 다른 자바스크립트 파일을 가져와 적용하기

inline에 자바스크립트 사용하기

  • HTML의 이벤트 속성들에는 자바스크립트 문법을 사용할 수 있다. (style 속성에는 CSS 문법을 사용할 수 있었던 것과 유사함)
  • HTML의 이벤트 속성들은 on으로 시작한다.
