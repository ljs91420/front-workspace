BOM (Browser Object Model)

  • 웹 브라우저에 직접 접근하여 기능을 활용할 수 있는 객체
  • DOM도 BOM의 내부에 존재하는 객체이다.(window.document)
  • window는 생략이 가능하다.(너무 당연해서)

BOM 내부의 여러 객체들

  • window : 웹 브라우저 최상위 객체
  • window.document : 현재 페이지의 HTML 구조를 담고 있는 객체
  • window.history : 웹 브라우저의 페이지 이동 내역을 담고 있는 객체
  • window.screen : 웹 브라우저가 표시되는 화면에 대한 정보
  • window.location : 웹 브라우저의 현재 위치(경로)에 대한 정보
  • window.navigator : 웹 브라우저 자체에 대한 정보
