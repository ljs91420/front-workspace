position property

  • 포지션이 설정되지 않은 다른 요소들을 무시하면서 마음대로 배치할 수 있는 속성
  • 요소 위에 요소를 겹칠 수 있다.
  • static : 포지션이 설정되지 않은 상태
  • absolute : 절대값으로 위치를 설정
  • relative : 원래 있던 위치로부터 상대적으로 위치를 설정
  • fixed : 고정적인 위치를 설정
  • sticky : 원래 자리에 있다가 스크롤이 내려갈 때 고정될 위치를 설정
  • 포지션 속성을 설정한 후 top, left, bottom, right 속성으로 위치를 정할 수 있다.

position: relative

  • absolute는 본인의 영역을 완전히 버리고 자기 위치를 찾아간다.
  • relative는 본인의 원래 영역을 차지한 채로 자기 위치를 찾아간다.

포지션이 설정된 부모 요소 내부에서의 위치 설정

  • 부모 요소의 position이 static이면 포지션이 설정된 요소는 부모 요소를 벗어나서 설정된다.
  • 부모 요소에 static이 아닌 다른 포지션이 설정되어 있다면 해당 부모 요소 내부에서 위치를 찾는다.

  • 스크롤이 있어야 제대로 동작하는지 확인할 수 있다.
