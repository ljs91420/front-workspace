하나의 테두리 속성에 여러 값을 부여하는 경우의 동작

  • 속성에 값을 하나만 주는 경우(모든 방향)
  • 속성에 값을 두 개 주는 경우(위/아래, 왼쪽/오른쪽)
  • 속성에 값을 세 개 주는 경우(위, 왼쪽/오른쪽, 아래)
  • 속성에 값을 네 개 주는 경우(위, 오른쪽, 아래, 왼쪽)
