반응형 웹 사이트

  • 화면(viewport) 크기에 따라 다른 스타일이 적용되는 웹 사이트
  • 사용자에게 화면 크기에 알맞는 최적화된 스타일을 제공할 수 있다.
  • @media 미디어종류 and 원하는범위

사용 가능한 미디어 종류

  • all : 모든 기기 타입
  • print : 출력 화면에서의 스타일 지정
  • screen : PC, 태블릿, 스마트폰 등 일반 기기
  • speech : 화면을 읽어주는 기기
