웹 페이지에 CSS를 적용하는 방법 3가지

1. inline CSS(해당 요소의 스타일 속성 사용하기)
- 해당 요소의 style 속성에 스타일을 직접 정의하는 방식
- CSS 적용 우선 순위가 가장 높다
2. internal CSS(파일 내부 CSS)
- 같은 파일 내부에 존재하는 style 태그를 사용하는 방식
- #id 선택자가 .class와 tagname보다는 적용 우선 순위가 높다.
- .class 선택자가 tagname 선택자보다는 적용 우선 순위가 높다. - 우선 순위가 같다면 더 나중에 나온 스타일이 앞에 있던 스타일을 덮어쓴다.
3. external CSS(파일 외부 CSS)
- 다른 .css 파일을 현재 웹 페이지로 불러서 사용하는 방식
- <link>를 사용해 외부의 .css 파일을 현재 페이지에 로드할 수 있다.
- 우선 순위는 internal CSS와 완전히 동일하다.
