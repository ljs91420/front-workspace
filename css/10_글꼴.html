font-family : 글꼴 설정

  • font-family 속성으로 해당 요소에서 사용하는 글꼴을 정의할 수 있다.
  • 기본적으로는 웹 페이지를 보는 사람의 PC에 설치되지 않은 글꼴은 사용할 수 없다.
  • 웹 페이지를 보는 사용자의 컴퓨터에 글꼴이 없는 경우에 대비하여 같은 크기를 지닌 여러 예비 글꼴을 설정할 수 있다.
  • 어떤 글꼴에 대한 예비용 글꼴을 선택할 때는 해당 글꼴의 계보를 따르는 편이다.
  • 접속한 사용자에게 글꼴을 설치하게 하기보다는 CDN을 사용해 온라인상에 존재하는 글꼴 파일을 참조하게 하는 편이다.
  • CDN : Content Delivery Network, 다른 회사들이 인터넷에 올려놓고 제공하는 이미지, 글꼴 등의 리소스들을 의미한다.
