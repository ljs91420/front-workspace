절대 길이 단위

  • 웹 브라우저(또는 다른 프로그램)의 크기가 변하더라도 항상 일정한 크기를 가지는 단위
  • px : 픽셀(화소)
  • mm : 밀리미터
  • cm : 센치미터
  • in : 인치

상대 길이 단위

  • 화면을 보는 프로그램(viewport)의 크기가 변하면 크기가 함께 변하는 단위
  • % : 부모 요소의 몇 %를 차지하는가(부모 요소의 크기가 변하면 변함)
  • em : 현재 요소의 font-size 속성에 비례한 크기 설정
  • rem : 현재 전체 페이지(html 태그)의 font-size 값에 비례한 크기 설정
  • vw : viewport width, 현재 화면 너비에 비례한 크기 설정(1vw는 현재 화면 전체 너비의 1%를 의미)
  • vh : viewport height, 현재 화면 높이에 비례한 크기 설정(1vh는 현재 화면 전체 높이의 1%를 의미)

10em
10rem
20vw
20vh
