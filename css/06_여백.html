margin

  • 해당 요소와 다른 요소 간의 거리를 결정하는 여백
  • border 바깥쪽에 존재하는 여백
  • margin-bottom과 margin-top이 만나는 곳에서는 둘 중 더 높은 값 하나만 적용된다.
  • margin을 auto로 설정하면 가운데로 정렬해준다.

padding

  • 요소 내부의 내용이 차지하는 영역(content-box)부터 border까지의 거리를 의미하는 여백
  • padding은 기본적으로 요소 전체의 크기에 영향을 미친다.

HTML Box Model

  • HTML은 모든 요소를 박스 형태로 구성한다.
  • 박스의 제일 바깥쪽 여백을 margin이라고 부르고, border, padding, content-box 순으로 요소를 구성하고 있다.
  • 요소의 전체 크기는 border의 굵기와 padding의 크기와 content-box의 크기를 더한 값이 된다.

box-sizing

  • whdth/height 값이 어떤 박스의 크기를 의미하는지 설정하는 속성
  • box-sizing: content-box는 width/height의 값이 content-box의 크기를 설정한다(기본 설정)
  • box-sizing: border-box는 width/height의 값이 요소의 전체 크기를 설정하게 되고 border-width, padding, content-box의 크기는 알아서 조절된다.
box-sizing: content-box;
width/height가 content-box의 크기를 결정한다.
box-sizing: border-box;
width/height가 border-box의 크기를 결정한다.(요소 전체의 크기를 결정한다.)
